# Typescript 类型 学习记录

 跟着[题目库](https://github.com/type-challenges/type-challenges)，学习一下`ts`类型

### 包管理工具

  `pnpm`

### 如何做题

1. 拉取题目到本地
2. 修改 `template.ts`，使得 `test-cases.ts` 不报错即可

## 知识点

- 定义变量：`infer`
- 取对象 T 的 key：`keyof T`
- 遍历数组 K：`[P in K]`
- 取数组 T 每一项：`T[number]`
- 数组中 Rest 的使用：`...`
- 取数组长度：`T['length']`
- 约束：`extends`
- 模板字符串：``` `${xxx}` ```
